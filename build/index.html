Readme

Сборка

Для сборки верстки используется 11ty в связке шаблонизатором Nunjucks.
Для обработки Css - postCss and csso.
Иконки собираются в спрайт. Все контурные элементы должны быть приведены к заливке.

Стили

Все цвета и размеры задаются css-переменными.
Есть глобальные и есть конкретные для темы (дефолтная, контрастная, темная).
Все цвета задаются в пространстве hsla (hue, saturation, lightness, alpha-channel).
Все размеры кратны 4рх и высчитываются от базового размера шрифта: 1rem = 16px. Изредка используется 10рх.
Для организации и наименования стилей используется методология БЭМ.
